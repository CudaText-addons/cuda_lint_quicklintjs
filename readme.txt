This is linter for CudaLint.
It adds support for JavaScript and "JavaScript Babel" lexers using "quick-lint-js" tool.
Tool homepage: https://quick-lint-js.com/install/cli/

How to install the tool:
Windows:
$ winget install quick-lint-js
macOS:
$ brew install quick-lint-js
Linux:
refer to the homepage. 

After installing make sure you can actually run it:
$ quick-lint-js --version
